# NekoMemo

NekoMemo 是一款背题软件，可从学习通网页提取题库，帮助背题练习喵。

## 功能特性

- **网页提取 (Fetcher)**：内置 WebView 浏览器，可访问教育平台并直接从页面源码中提取题目。
- **题库管理**：组织并管理多个题库。支持查看详情、追踪进度以及删除不需要的题库。
- **练习模式**：通过简洁、无干扰的界面，对提取的题库进行自我测试。
- **导入/导出**：通过 JSON 格式备份题库或与他人分享。
- **Material 3**

## 使用说明

1. 在 Fetcher 登录学习通
2. 找到课程的作业详情页面
3. 点击右下角按钮，提取题目
4. 保存到题库
